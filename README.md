## Getting Started
1. open terminal
```
git clone https://github.com/crazy-hour-ai/trash-talk-generator.git
```
2. Initialize project
```
npm install
```
3. Start project on development mode
```
npm run dev
```
4. Start the project on browser by 
```
http://localhost:3000 
```
